Ce site web est un outil de saisie et de représentation de données ayant pour but de faciliter votre travail de recherche dans les archives de Nantes de la première guerre mondiale.

Comment ça marche ?

Fonctionement :

  • Remplissez une ou plusieurs fois le formulaire pour obtenir une représentation des données sous forme de graphique.
    Attention, tous les champs doivent être remplis.

  • Ou bien cliquez directement en haut à droite sur "Graphiques" pour une représentation directe des
    données sans saisie.
